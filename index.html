¿Qué es Control de Versiones?

El control de versiones es la gestión de cambios a documentos, programas de computadora y otras colecciones de información. Cada cambio se denomina "versión".

Comandos Básicos de Git

Inicializar Repositorio

git init

Crea un nuevo repositorio Git en el directorio actual

Agregar Archivos

git add archivo.txt

Agrega archivos al área de staging

Confirmar Cambios

git commit -m "Mensaje"

Guarda los cambios con un mensaje descriptivo

Subir Cambios

git push origin main

Envía los cambios al repositorio remoto

Descargar Cambios

git pull origin main

Obtiene los últimos cambios del repositorio remoto

Crear Rama

git branch nueva-rama

Crea una nueva rama para desarrollo paralelo
